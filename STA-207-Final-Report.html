Originally, STAR planned to keep students in the same classroom type from kindergarten through third grade, but that didn’t last. After parents raised concerns, noted in Hanushek (1999), the study was modified so that at the start of first grade, all students in regular‑size classes—whether or not they had an aide—were randomly reassigned to either a regular class with an aide or one without. Students who began in small classes weren’t part of this reshuffling and stayed in small classes for the rest of the study.

Beyond this official change, schools didn’t always follow the assignment rules in practice. Some, especially smaller ones, made informal adjustments once the year began. These weren’t parent-driven; they were practical decisions by administrators or teachers trying to manage real‑world constraints. Schools moved students to balance class sizes, address classroom dynamics, and etc. None of this was part of the experimental protocol, but it still mattered because it introduced nonrandom noncompliance. Finn & Achilles (1999) note that this weakens the clean randomization and can bias causal estimates.

# 1. PREPARE THE DATA
# ==========================================
coef_data <- tidy(hlm_dummy_2, effects = "fixed", conf.int = TRUE) %>%
  filter(term != "(Intercept)")

coef_clean <- coef_data %>%
  mutate(
    Effect_Group = ifelse(str_detect(term, ":"), "Interaction Effects", "Main Effects"),
    Clean_Term = case_when(
      term == "freelunch" ~ "Free/Reduced Lunch", 
      term == "small_class1" ~ "Small Class",
      term == "minority1" ~ "Minority Student",
      term == "grade2" ~ "Grade 2",
      term == "grade3" ~ "Grade 3",
      term == "small_class1:minority1" ~ "Small Class × Minority",
      term == "small_class1:grade2" ~ "Small Class × Grade 2",
      term == "small_class1:grade3" ~ "Small Class × Grade 3",
      term == "minority1:grade2" ~ "Minority × Grade 2",
      term == "minority1:grade3" ~ "Minority × Grade 3",
      term == "small_class1:minority1:grade2" ~ "Small Class × Minority × Grade 2",
      term == "small_class1:minority1:grade3" ~ "Small Class × Minority × Grade 3",
      TRUE ~ term
    )
  ) %>%
  arrange(Effect_Group, estimate) %>%
  mutate(Clean_Term = factor(Clean_Term, levels = unique(Clean_Term)))

# ==========================================
# MAIN EFFECTS ONLY
# ==========================================
main_data <- coef_clean %>% filter(Effect_Group == "Main Effects")
